Overview Pricing

CoveredCI.com allows you to find the bugs hiding in your Web services and produces interactive reports in minutes. On each git push our automated property-based testing engine starts your service, generates specially-tailored JSON data and runs real-world test cases all simply from your JSON APIs documentation. Invalid data, security vulnerabilities, wrong HTTP logic and crashes are all reported along with a minimal test case.

Features

  • Detects real-world bugs that no amount of unit testing would uncover.
  • Detects CWEs and OWASP Top 10 vulnerabilities before they impact your users.
  • Runs hundreds of tests per push, and hundreds of thousands during the night.
  • Supports standard API documentation formats such as OpenAPI, API Blueprint and others
  • Dramatically increases code coverage
  • Generates API clients in most general-purpose programming languages
  • Generates and deploy documentation websites for you
  • Generate easy-to-read report explaining issues and displaying relevant data in a high level manner, to ensure readability by the widest (and potentially nontechnical) audience.
